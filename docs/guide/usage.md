# Usage

- **Home**: Scan or search to open storage units, containers, items, or projects.
- **Inventory**: Browse and drill into locations.
- **Projects**: BOM lines, CSV import/export, Mouser/TME shopping lists from project or global order list.
- **Order list**: Low stock and project gaps; refresh supplier prices when keys are configured.
- **Settings**: Categories, scanner status, backup, label sidecar, printable command sheet. Scanner status reflects serial hardware when `SCANNER_PORT` is set — see **[Hardware scanner](./hardware-scanner.md)**.

Command codes (`CMD:*`, `QTY:*`) match the desk command sheet served at `/api/labels/command-sheet`.
